Functions on : Router
Works on : 
Technology in which multiple routable IP addresses are converted into special range of IP Addresses.
Usually on 1 to 1 basis.

## Translation Modes
1. Static Translation
	A block of external addresses are converted to same size of internal addresses.
2. Dynamic Translations
	Large Number of Internal Users share a single pool of external addresses
3. PAT
	PAT Allows a single public IP Address to be used by many hosts on the Private network which is usually LAN.
